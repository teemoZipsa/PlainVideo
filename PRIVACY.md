# PlainVideo privacy policy

Effective date: 2026-07-21

PlainVideo is a local video player. It does not require an account and does not
contain advertising, analytics, telemetry, or tracking.

## Data handled by the app

PlainVideo opens media and subtitle files that you choose on your Windows
device. Playback, subtitle selection, screenshots, preferences, window
placement, and resume history are processed locally. PlainVideo does not upload
the contents, names, paths, metadata, or viewing history of those files.

Preferences and resume history are stored only in the current Windows user
profile. Uninstalling or clearing the app's local data may remove this
information.

## Network access

The current PlainVideo release does not provide network streaming, cloud sync,
accounts, recommendations, crash reporting, or automatic telemetry. Links to
the source code, support, and third-party license information open in the
user's default web browser only when the user chooses them.

## Third parties

PlainVideo includes open-source playback components. Their licenses and source
information are included with the distributed package. These components do not
change PlainVideo's data-handling policy.

## Contact

For privacy questions, open an issue in the public
[PlainVideo repository](https://github.com/teemoZipsa/PlainVideo/issues).

If PlainVideo's data handling changes, this policy and the Microsoft Store
listing will be updated before the changed version is published.
