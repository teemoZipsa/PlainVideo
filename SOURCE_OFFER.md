# PlainVideo 0.2.1 corresponding source offer

PlainVideo 0.2.1 uses the same reviewed dynamically loaded library build as
0.1.0, including mpv, FFmpeg, libplacebo, GLib, FriBidi, Graphite2,
libiconv, and other open-source projects. The complete source and build
material retained for the exact distributed DLLs is available at:

https://github.com/teemoZipsa/PlainVideo/releases/download/v0.1.0/PlainVideo-0.1.0-corresponding-source.tar.zst

The PlainVideo application source corresponding to version 0.2.1 is available
at:

https://github.com/teemoZipsa/PlainVideo/tree/v0.2.1

The bundle contains the four direct source archives and pinned submodules,
the exact official MSYS2 source-only packages for all 14 transitive runtime
packages, the package recipes and patches, build configuration evidence, and
SHA-256 inventory. PlainVideo loads the libraries as replaceable side-by-side
DLLs and does not rename or conceal them.

SeonkyuIM will keep this corresponding source available for at least three
years after the last distribution of PlainVideo 0.2.1. If the download becomes
unavailable, request an equivalent machine-readable copy through
https://github.com/teemoZipsa/PlainVideo/issues. Any physical-media charge will
not exceed the reasonable cost of performing that distribution.

This offer covers the open-source library material required by the applicable
licenses. Each component remains under its own license; see
`THIRD_PARTY_NOTICES.md` and the `licenses` directory in the app package.
