# PlainVideo support

PlainVideo is a borderless local video player for Windows 10 and Windows 11.
The Microsoft Store and portable releases currently target x64 desktop devices.

## Basic controls

- Open a video: drag it onto the window or press `Ctrl+O`.
- Play or pause: click the video or press `Space`.
- Fullscreen: double-click the center of the video or press `F`.
- Seek: use Left/Right, double-click the left or right side, or hover over the seek bar for a thumbnail preview. Hold `Shift` with Left/Right for a larger step.
- Volume: use Up/Down or the mouse wheel; press `M` or the speaker button to mute or restore sound.
- Subtitles: use the CC button for direct on/off control or the right-click menu for track selection. Drop one subtitle file onto an active video to attach it without replacing playback.
- Subtitle timing: use `Ctrl+[` for 0.1 seconds earlier, `Ctrl+]` for 0.1 seconds later, and `Ctrl+\` to reset.
- Audio tracks and playback speed: use the right-click menu or their keyboard shortcuts.
- Media information: press `Tab` without interrupting playback.
- Save a screenshot: press `S`; the result is saved in the Windows Pictures library.
- Exit fullscreen: press `Esc`.

PlainVideo automatically discovers an external subtitle when it is beside the
video and uses a matching base name. Partially watched local files resume from
their saved position; use the right-click menu to restart from the beginning.

## Getting help

Search existing reports or create a new issue at
[github.com/teemoZipsa/PlainVideo/issues](https://github.com/teemoZipsa/PlainVideo/issues).
Include the PlainVideo version, Windows version, the media container and codec
when known, and the steps that reproduce the problem. Do not attach private or
copyrighted videos, personal file paths, account details, or other sensitive
information.

Source code and license notices are available in the
[PlainVideo repository](https://github.com/teemoZipsa/PlainVideo).
