## English — en-US

### Product name

PlainVideo

Reserved in Partner Center on 2026-07-21.

### Short description

A calm, borderless local video player that keeps controls out of the way.

### Description

PlainVideo puts the video first. Open a local file and start watching in a borderless Windows window, with familiar controls that appear only when you need them.

Click to play or pause, double-click for fullscreen, preview a position by hovering over the seek bar, and use the arrow keys to seek or change volume. Toggle subtitles directly from the transient playback bar, choose a track from the context menu, or drop a subtitle onto an active video. PlainVideo automatically discovers matching external subtitles, supports 0.1-second subtitle timing correction, resumes partially watched files, and remembers volume and a safe physical window position across monitors.

PlainVideo is designed for local playback. It has no ads, accounts, recommendations, or telemetry. The source code and third-party license information are public.

Press `Tab` for compact media details without interrupting playback, or press `S` to save a playback screenshot to the Windows Pictures library.

PlainVideo lists only formats and hardware paths verified against its exact packaged playback runtime.

### What's new in version 0.2.1

The volume and mute control is cleaner and easier to read in both sound states. Drop one subtitle file onto a playing video to attach it without replacing playback, then correct subtitle timing in 0.1-second steps or reset it from the context menu and keyboard.

### Features

- Borderless, content-first local playback
- Click play/pause and double-click fullscreen
- Compact controls that disappear after inactivity
- External subtitle discovery and track selection
- Subtitle-file drop attachment and 0.1-second timing correction
- Seek-bar thumbnail previews and automatic resume
- Playback screenshots and richer media information
- Multi-monitor physical window placement
- Korean and English interface
- No ads, accounts, or telemetry
- Open-source code and explicit third-party notices

## 한국어 — ko-KR

### 제품 이름

PlainVideo

2026-07-21 Partner Center에서 이름을 예약했습니다.

### 간단한 설명

조작부는 조용히 숨기고 영상에 집중하는 테두리 없는 로컬 동영상 플레이어입니다.

### 설명

PlainVideo는 영상을 화면의 중심에 둡니다. 로컬 파일을 열면 테두리 없는 Windows 창에서 바로 재생하며, 익숙한 조작부는 필요할 때만 나타납니다.

클릭으로 재생하거나 일시정지하고, 더블 클릭으로 전체화면을 전환하며, 탐색바에 마우스를 올려 해당 시점의 썸네일을 미리 볼 수 있습니다. 방향키로 탐색과 볼륨을 조절하고, 잠깐 나타나는 재생 바에서 자막을 바로 켜고 끄거나 우클릭 메뉴에서 트랙을 고를 수 있습니다. 재생 중인 영상에 자막 파일 하나를 놓아 추가하고 0.1초 단위로 싱크를 맞출 수도 있습니다. 같은 이름의 외부 자막을 자동으로 찾고, 일부 재생한 파일의 위치와 볼륨, 모니터별 안전한 물리 픽셀 창 위치를 기억합니다.

PlainVideo는 로컬 재생을 위한 앱입니다. 광고, 계정, 추천 피드, 텔레메트리가 없습니다. 소스 코드와 제3자 라이선스 정보도 공개합니다.

재생을 멈추지 않고 `Tab`으로 간결한 미디어 정보를 확인하고, `S`로 재생 화면을 Windows 사진 라이브러리에 저장할 수 있습니다.

실제 패키지에 포함된 재생 런타임으로 검증한 형식과 하드웨어 경로만 지원 범위로 표시합니다.

### 버전 0.2.1의 새로운 기능

소리와 음소거 상태 모두에서 볼륨 조작부가 더 간결하고 읽기 쉬워졌습니다. 재생 중인 영상에 자막 파일 하나를 놓으면 영상을 바꾸지 않고 자막만 추가하며, 우클릭 메뉴와 키보드로 자막 싱크를 0.1초씩 조절하거나 초기화할 수 있습니다.

- 재생 화면 스크린샷과 확장된 미디어 정보
