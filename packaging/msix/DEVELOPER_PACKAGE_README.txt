PlainVideo developer MSIX proof

This package uses the local LGPL libmpv candidate only to test MSIX layout,
installation, activation, and Windows integration. The candidate is not
approved for redistribution and this package must not be uploaded to Partner
Center, shared, or published.

Only a build explicitly created with the Store-upload gate enabled after
docs/STORE_RELEASE_STATE.json marks the runtime and release eligible may be
considered for upload.
